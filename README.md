## 📌 What is this?

**Mini Compressor** is a powerful all-in-one tool for game repacking and file compression. Think of it as a *Swiss Army Knife* — it combines multiple compression engines into a simple GUI, so you don’t have to deal with command-line tools.

It’s been tested over time and proven stable. The Masked Compression feature offers excellent compression ratios (if you’re willing to wait).

---

## 🚀 Features

- Masked Compression v3.0
- Self-extracting archives (SFX support)
- Custom background music & wallpaper
- Built-in auto-updater *(Help → Check for updates)*
- Multi-threaded processing (uses your CPU efficiently)
- Game-specific compression optimization (textures, audio, etc.)

## ⚙️ How to Use

1. Open `Launcher.exe`
2. Select your game folder
3. Choose a compression method  
   - **Masked** → Best compression  
   - **Standard** → Faster processing
4. Enable **Create SFX** if needed
5. Click **Compress** and wait (can take hours)

---

## ⚠️ Important Notes

- ❌ **Do NOT use DiskSpan with Masked Compression**  
  → This will corrupt your files.

- 💡 **PDB files (debug symbols):**  
  Use **SREP filters** for best results.

- 🖼 **PNG files:**  
  Already compressed — avoid heavy methods.

## ❓ FAQ

**Q: A compressor failed. What should I do?**  
A: The tool uses third-party compressors. Errors may come from them.  
- Take a screenshot  
- Backup your data  
- Try another compression method  

---

**Q: Why can't I use DiskSpan with Masked Compression?**  
A:  
- DiskSpan splits files  
- Masked modifies data streams  
→ Together, they cause corruption
