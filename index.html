📌 What is this?

Mini Compressor is an all-in-one tool for game repacking and file compression. It combines multiple compression engines into a simple GUI so you don’t have to deal with command-line tools.

🚀 Features

📦 Masked Compression v3
⚡ Multi-threaded performance
🎮 Game optimized compression
🎵 Custom music & wallpaper
🔄 Auto updater
📁 SFX archive support

💻 System Requirements

Minimum:

  • 4GB RAM
  • 320GB Storage
  • Windows 7/8/10 x64

Recommended:

  • 8GB+ RAM
  • 512GB+ Storage
  • Windows 10/11 x64

⚙️ How to Use

  1. Open Launcher.exe
  2. Select your game folder
  3. Choose compression method
  4. Enable SFX if needed
  5. Click Compress and wait

⚠️ Important Notes

  • Do NOT use DiskSpan with Masked Compression
  • Use SREP filters for PDB files
  • PNG files are already compressed

🎨 Customization

Music: Options → Audio Settings
Wallpaper: Options → Appearance
