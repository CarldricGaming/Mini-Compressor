What is this?

Mini Compressor is an all-in-one tool for game repacking and file compression. It combines multiple compression engines into an easy-to-use GUI, eliminating the need for complex command-line tools.

Features

📦 Masked Compression v3
⚡ Multi-threaded performance
🎮 Game optimized compression
🎵 Custom music & wallpaper
🔄 Auto updater
📁 SFX archive support

Preview

How to Use

  1. Open Launcher.exe
  2. Select your game folder
  3. Choose compression method
  4. Enable SFX if needed
  5. Click Compress and wait
